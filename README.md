# Proyecto-2DAW
Repositorio para el proyecto conjunto de fin de Proyecto Intermodular

---

# Proyecto Angular

Este proyecto fue generado con [Angular CLI](https://angular.io/cli) versión 20.3.10.

## Instalación

```powershell
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo y abrir la app en el navegador:

```powershell
npx ng serve --open
```

La aplicación estará disponible en [http://localhost:4200](http://localhost:4200).

## Linter

Para analizar la calidad del código (después de configurar ESLint):

```powershell
npx ng lint
```

## Construcción

Para generar una versión de producción:

```powershell
npx ng build
```

## Estructura principal
- `src/` — Código fuente de la aplicación
- `angular.json` — Configuración del proyecto Angular
- `package.json` — Dependencias y scripts

## CI/CD

Este proyecto puede integrarse con GitHub Actions para ejecutar lint y build automáticamente en cada push/pull request.
